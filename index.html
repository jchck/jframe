jFrame.css v0.0.2

delightfully simple scalable css

Getting Started

jFrame is an object oriented CSS toolkit designed to help you and your team write better and easier to maintain CSS by writing less of it. This is a library of single purpose CSS classes focused on extensibility, flexibility, and reuseability. Don't get lost in the cascase, make better sense of it.

Note: This project is still very much in constant development, if you find any bugs (as you likely will) please submit a pull request.

src/_media-queries.css

All media queries are written mobile first. Don't be shy about adjusting these (or adding more) to match your content, not the other way around.

The Gulp-PostCSS build process will collect and order all queries so you just put them where ever makes sense to you. If a computer can do it why should we?

@custom-media --screen-xs (min-width: 30em);
@custom-media --screen-sm (min-width: 60em);
@custom-media --screen-md (min-width: 90em);
@custom-media --screen-lg (min-width: 120em);

White Space

These margin and padding utility modules can greatly reduce the size of CSS file sent to the browser. Attach these classes to any element to adjust white space as appropriate.

src/_space-margin.css

Classes are also scoped to specific media queries to make adjustments at various screen widths a breeze.

First, negative margins are provided along the X (left to right) axis only. Use these to adjust for container padding inside the grid. Next, margin classes allow us to reset all margins or add them based on the scale defined in src/_variables.css. And finally we have a left/right margin auto class to center bloc elements within their parent container.

.css

src/_space-padding.css

As with margin classes, these padding classes are also scoped to media query. Note how every padding class comes with definations for sizes along both the X and Y axis. This is to encourage visual rythem through out.

.css

/* Non Scoped */
.p0{ padding: 0; }

.p1{ padding: var(--space-1); }
.px1{ padding-right: var(--space-1); padding-left: var(--space-1); }
