Typography

First we set the base font-family, line-height, font-size, and we create our type scale.

the html

<p>Lorem ipsum...</p>

<h1>heading 1<h1>
<h2>heading 2</h2>
<h3>heading 3</h3>
<h4>heading 4</h4>
<h5>heading 5</h5>
<h6>heading 6</h6>

<ol>
	<li>First</li>
	<li>Second</li>
	<li>Third</li>
</ol>

the css

body{
	font-family: var(--font-family);
	line-height: var(--line-height);
	font-size: var(--body-font-size);
}

h1, h2, h3, h4, h5, h6{
	font-family: var(--heading-font-family);
	font-weight: var(--heading-font-weight);
	line-height: var(--heading-line-height);
	margin-top: var(--heading-margin-top);
	margin-bottom: var(--heading-margin-bottom);
}

h1, .h1{ font-size: var(--h1-font-size); }

h2, .h2{ font-size: var(--h2-font-size); }

h3, .h3{ font-size: var(--h3-font-size); }

h4, .h4{ font-size: var(--h4-font-size); }

h5, .h5{ font-size: var(--h5-font-size); }

h6, .h6{ font-size: var(--h6-font-size); }

.measure{ max-width: var(--measure-width); }

p{
	margin-top: var(--p-margin-top);
	margin-bottom: var(--p-margin-bottom);
}

dl, ol, ul{
	margin-top: var(--list-margin-top);
	margin-bottom: var(--list-margin-bottom);
}

Then we add some utility classes for type alignments, weights, and styles.

the html

<h1 class="type-left">heading 1<h1>
<h2 class="type-center">heading 2</h2>
<h3 class="type-right">heading 3</h3>
<p class="type-justify">Lorem ipsum...</p>

<span class="bold">Bold</span>
<span class="regular">Regular</span>
<span class="italic">Italic</span>
<span class="uppercase">Uppercase</span>
<span class="lowercase">Lowercase</span>
<span class="capitalize">Capitalize</span>

the css

/* Alignments */
.type-left{ text-align: left; }
.type-center{ text-align: center; }
.type-right{ text-align: right; }
.type-justify{ text-align: justify; }

/* Weights & Styles */
.bold{ font-weight: var(--font-weight-bold); }
.regular{ font-weight: normal; }
.italic{ font-style: italic; }
.uppercase{ text-transform: uppercase; }
.lowercase{ text-transform: lowercase; }
.capitalize{ text-transform: capitalize; letter-spacing: .1em; }

Finally, we define code block styles.

the html

<pre>
	<code>
	...
	</code>
</pre>

the css

pre{
	max-height: var(--pre-max-height);
	overflow-x: scroll;
}

code{
	font-family: var(--code-font-family);
	font-size: var(--code-font-size);
}
